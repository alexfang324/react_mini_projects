This is a React.js app on making a several basic UI widget using the Hook system. The app also includes the concept of throttling and debouncing of user input and the concept of event bubbling when building a dropdown menu with several onClick handlers. Moreover, this app uses a custom Router built from an useEffect hook and self-written Router and Link components. You can start this app with Node.js by first downloading the dependencies using the command "npm install" and then start the app with "npm start".
